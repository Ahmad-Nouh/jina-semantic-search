# jina-semantic-search
Semantic Search Using Jina


# Environment
- Python 3.9.2

# Requirements
- click==8.0.1
- pandas==1.3.1
- jina==2.1.5
- python_daemon==2.3.0

# Running
- Running the Flows:

    ``
        python app.py -t flows
    ``
    
- Query your data:
    
    ``
        python app.py -t client
    ``
